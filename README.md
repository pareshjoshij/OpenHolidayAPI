## üí° How to Use

1.  **Select Region:** Choose your target Country/Region from the dropdown.
2.  **Set Locale:** (Optional) Define the language or locale specifics.
3.  **Fetch Data:** Hit the blue button to retrieve the holiday list.
4.  **Export or Code:**
    * *For Users:* Click **Download .ICS** to add to your calendar.
    * *For Devs:* Copy the generated code from the **API Console** panel.
